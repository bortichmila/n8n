# Use Cases

- L&D-бот для онбординга: отвечает на вопросы новичков, отправляет чек-листы, собирает фидбек.
- Бот-консьерж для ивента: программа, расписание, ответы на FAQ.
- Контент-бот: генерирует варианты постов, согласует и отправляет в канал.
- Лид-бот: собирает заявки, отправляет в CRM и уведомляет менеджера.
